Steam Bot - Part 1 2019-03-28|Nodejs-JaveScript-Steam-Chat Bot

Introduction

Many games in steam use steam inventory to store their items, with this inventory system, player can trade their item with other player more easily and convenient.

Steam bot can sell/buy items automatically, so many people build some bot and start their own business.

There are serval place you can find the bot

  • backpack.tf Classifieds
  • scrap.tf
  • cs money
  • etc.

In this series, I will teach how to build a sample bot.

System Requirement

Before we start, There are some thing need to be installed.

  • Node.js and NPM
  • steam-user
    • node js package for simulating Steam client
    • npm install steam-user
  • steam-totp
    • node js package for getting Steam Guard code
    • npm install steam-totp

Our First Bot

The code below is our first bot

  1. Create a file steambot.js
  2. Copy the code and paste in your file
  3. Replace YOUR_ACCOUNT_HERE and YOUR_PASSWORD_HERE to your bot’s account and password
  4. Run the script using node steambot.js commend
  5. Enter your Steam Guard Code

After these steps, you will see your bot online and start playing Team Fortress 2.

Modules

At the top of our script, we import steam-user modules and create a instance client for simulating steam clinet.

User Setting

logOnDetails is parameter for steam client login, if you want to know more detail this object just check here, there are more information you can read.

Login and LoggedOn Event

We use our logOnDetails as parameter to logOn, it will login Steam account base on the account you set before.

Finally, We register a event listener for loggedOn event, and give it a callback function.

This event is emited when Steam have logged in, then execute our callback function.

In this callback funcion, we show a message to notify us the bot is ready, and set steam persona state to Online, set playing game to Team Fortress 2.

440 is AppID of Team Fortress 2

All personaState can find here.
If you are not setting persona state, the default state is offline

The parameter for this function is AppID or string, if you give a string it will show you are playing Non-Steam game and game name will be the string you give.

e.g., I give a string like this Selling Item, it will show playing non-Steam game Selling Item.
